# Feature Branch Workflow

- Create a feature branch before making any changes
- Branch from the main branch (main/master)
- Make meaningful commits during work
- Run tests, type checker, and build before pushing
- Push the branch and create a pull request — never merge directly into main
- Delete the branch after the PR is merged
