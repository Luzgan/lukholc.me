Tableau Workbook Explorer

Project Overview

The Tableau Workbook Explorer is a powerful tool designed to help Tableau developers and analysts understand the complex relationships between fields in their workbooks. By uploading local Tableau files, users can visualize field dependencies and gain insights into their calculation structures.

Key Features

  • • Upload and parse local Tableau workbooks (.twb files) - its kept on your local machine, nothing is uploaded to the server
  • • Interactive dependency graphs using React Flow
  • • Information about different fields, also with ability to see if they are hidden or not

Technical Stack

JavaScript React React Flow XML Parser

Problem Solved

Tableau workbooks often contain complex calculated fields with multiple dependencies. Understanding these relationships is crucial for:

  • • Debugging calculation errors
  • • Optimizing workbook performance
  • • Documenting field relationships
  • • Onboarding new team members
  • • And most importantly, finding out the scope of deleting given field

Development Process

The project involved parsing Tableau's XML-based workbook format, extracting field definitions and dependencies, and creating an intuitive visual interface using React Flow for interactive node-based diagrams. Additionally, because of lack of documentation, I had to reverse engineer the Tableau API to understand the structure of the workbook and provide tests on different files to make sure the app is working as expected.
